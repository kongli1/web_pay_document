# 优惠交易平台

## 场景描述

### 退货

用户对买到的商品因各种原因想要退还商品返回本金，可以发起本操作

**参与者：**  买方（平台用户）、卖方（平台用户）

**自动退货**

**假设：** 卖方售卖商品支持自动退货、买方购买了商品且商品未打开、未到有效期   
**约束：** 买方已经完成购买商品流程（对订单支付完成），并进入未完成订单列表，可看到“申请退货”操作，如有其他操作会到其相应的页面  
   
- 场景1： 
       - 卖家无需确认退货 
        买方点击“申请退货”操作，卖方无需确认就可以完成退货流程（钱返还给买方），同时收到系统提示的退货信息
